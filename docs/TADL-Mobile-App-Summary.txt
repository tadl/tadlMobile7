TADL Mobile
One-page App Summary

TADL Mobile is a patron-facing mobile app for Traverse Area District Library.
It gives cardholders a faster, more focused mobile experience for the tasks they
use most often: checking account activity, searching the catalog, placing and
managing holds, browsing events, reading library updates, and finding accurate
branch information.

Core patron features
- Account access for checkouts, holds, fines, lists, saved preferences, and
  digital library card display
- Catalog search with filters, ISBN barcode scanning, and item detail modals
- Hold placement and hold management, including suspend, cancel, and pickup
  location changes
- Event browsing with detailed views and add-to-calendar support
- Newsletter feed and webcams feed managed through remote JSON endpoints
- Location pages with weekly hours, service exceptions, and navigation/contact
  shortcuts

Operational/technical highlights
- Built with Ionic, Angular, and Capacitor for iPhone and Android deployment
- Uses Aspen Discovery APIs for search, account actions, and grouped work data
- Uses feed proxies for app-managed content like newsletters and webcams
- Applies local caching to improve perceived speed and reduce repeat loading
- Supports theme preference, link-handling preference, and other account-level
  settings inside the app

Recent improvements
- News was replaced by a feed-backed Newsletter experience
- Webcams were moved to a feed-backed page with safer external playback
- Location closures and modified hours now use consistent Eastern-time handling
- Home page can surface service alerts for affected locations
- Item detail views gained stronger hold-state handling and clearer list feedback
- Small-screen layouts and contrast were improved across several pages

Current release posture
The app is in a polished, release-oriented phase. Most recent work has focused on
real-world patron flows, interface consistency, caching behavior, accessibility,
and operational reliability rather than major new feature development.
